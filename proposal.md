# 提案：创建知识竞赛静态网页PPT

## 为什么
用户需要一个可以在本地浏览器中直接打开的纯静态网页PPT，用于知识竞赛。该PPT需要支持：
- 通过JSON文件管理竞赛题目
- 可自定义背景图和文字配色
- 包含导航功能（全屏、返回首页、返回选题页）
- 三页结构：首页、选题页、题目页

## 什么改变
这是一个新功能项目，从零开始创建一个完整的静态网站应用。

### 核心功能
1. **首页**：展示竞赛名称，点击空白处跳转到选题页
2. **选题页**：显示所有题目序号，区分已选/未选状态（不同颜色）
3. **题目页**：显示题目内容，30秒倒计时，倒计时结束变红色，包含"显示答案"按钮
4. **导航条**：全屏按钮、返回首页按钮、返回选题页按钮
5. **配置系统**：通过JSON文件管理题目和样式配置

### 技术架构
- 纯静态网站（HTML + CSS + JavaScript）
- 无服务器依赖，可直接在浏览器中打开
- JSON配置文件用于题目数据和样式设置
- 响应式设计，适配不同屏幕尺寸

## 影响
- 创建新的项目目录结构
- 生成HTML、CSS、JavaScript和JSON配置文件
- 提供完整的使用文档和配置说明

### 文件结构
```
知识竞赛PPT/
├── index.html          # 主页面
├── styles.css         # 样式文件
├── script.js          # 交互逻辑
├── config.json        # 题目和配置数据
└── README.md          # 使用说明
```

### 配置文件示例
```json
{
  "contest": {
    "title": "知识竞赛",
    "backgroundImage": "background.jpg",
    "textColor": "#333333",
    "selectedColor": "#ff6b6b",
    "unselectedColor": "#51cf66"
  },
  "questions": [
    {
      "id": 1,
      "question": "题目内容",
      "answer": "答案内容",
      "selected": false
    }
  ]
}
```

## 成功标准

2. 通过修改config.json可以更新题目和样式
3. 所有导航功能正常工作
4. 倒计时功能准确（30秒）
5. 题目状态正确显示和更新

## 风险与注意事项
- 确保所有浏览器兼容性（使用标准HTML5/CSS3/JavaScript）
- 图片路径使用相对路径，确保在不同环境下正常显示
- JSON文件格式需要严格验证，避免解析错误
